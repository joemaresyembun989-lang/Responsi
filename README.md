1.Deskripsi singkat project :
FRAMESELF Flask adalah sebuah website portfolio pribadi yang dibuat menggunakan Python Flask dan MySQL. Website ini menampilkan profil, daftar skills dan proyek,serta foto. Selain halaman publik, terdapat halaman admin yang memungkinkan pemilik untuk mengelola data profil, skills, dan proyek melalui fitur CRUD (Create, Read, Update, Delete). Website ini dirancang untuk memberikan tampilan portfolio yangTampil secara profesional, menarik untuk digunakan, dan mudah diperbarui kapan saja.

2.Cara menjalankan aplikasi :
1. Pastikan Python dan MySQL sudah terpasang, serta MySQL server sedang berjalan.
2. Buat database dengan nama “responsi”, lalu jalankan script SQL project untuk membuat tabel users, skills, dan projects.
3. Pastikan foto profil tersimpan di folder static/images/foto.jpg.
4. Instal semua library yang dibutuhkan, yaitu Flask dan Flask-MySQLdb.
5. Jalankan aplikasi dengan perintah untuk mengeksekusi file utama (app.py).
6. Setelah aplikasi berjalan, buka browser dan akses alamat [http://127.0.0.1:5000/](http://127.0.0.1:5000/) untuk halaman publik atau [http://127.0.0.1:5000/login](http://127.0.0.1:5000/login) untuk halaman admin.
